## Titulo
Proyecto Programa Salón Arcade

## Descripción
El programa tiene un menú principal que se divide en tres submenús uno para jugadores, otro para máquinas y otro
para estadísticas y listados. En el menú principal se redirije a dichos submenús y también permite jugar una partida.
Hay tres clases principales en el programa Jugador, Máquina Arcade y Sala Recreativa/Arcade.
Durante una partida se llaman a varios métodos y se verifica que no hay problemas, si algo falla se lanza una
excepción específica.
